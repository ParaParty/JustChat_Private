# JustChat 聊天室

## 概述
- 一个用来实现 ```Sponge服务端``` 和 ```QQ群``` 双向同步聊天的工具。

## 特性
- 直接运行在网络层，资源消耗低。
- 服务端架设在```QQ机器人```处，让面板服务器亦可使用本插件。
