# 报文结构
## 总述
- 报头 7字节
	- 0x11 0x45 0x14
	- 4字节 表示包体长度 0x????????
- 报体 根据报头取得的报体长度获取
	- json结构
	
## 报体结构
- 报体为json结构，解析后可得到
	1. int version
		- 表示数据包版本
		- 当前版本 : ```version = 0```
	2. int type
		- 表示数据包类型	
- 普通消息 ```type = 0```
	1. string sender
		- 将原字符串内容按UTF-8编码后再进行Base64编码的字符串
		- 表示消息发送者
	2. string content
		- 将原字符串内容按UTF-8编码后再进行Base64编码的字符串
		- 表示消息内容
